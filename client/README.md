## Directory Structure

```text
client/
├── src/
│   ├── App.jsx         # Unified Application Engine
│   └── index.css       # Global Design System & Animations
├── public/             # Static Assets & Images
├── index.html          # Entry Point
├── package.json        # Dependencies & Scripts
└── tailwind.config.js  # Theme Configuration
```

## Core Features

- **Unified Component Architecture**: All logic and components are consolidated within `App.jsx` for maximum efficiency.
- **Dynamic Heritage Map**: Integrated Leaflet map with custom marker clustering and interactive spot previews.
- **Premium Design System**: Tailored dark mode with glassmorphism effects and smooth micro-animations.
- **Adaptive UI**: Fully responsive layouts optimized for both mobile explorers and desktop administrators.

## Development

1. Install local dependencies:
   ```bash
   npm install
   ```
2. Configure environment:
   Copy `.env.example` to `.env` and provide your Supabase credentials.
3. Start development server:
   ```bash
   npm run dev
   ```
